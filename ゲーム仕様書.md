# ゲーム仕様書: ぼくはりそうのちちおや

## 1. 基本情報
* **タイトル:** ぼくはりそうのちちおや
* **ジャンル:** Reignsライク・カードスワイプ型ギャグシミュレーション
* **ターゲット:** 手軽にブラックジョークや風刺を楽しみたいプレイヤー
* **開発エンジン:** Godot Engine 4.x
* **コアコンセプト:** 「息子のため」という名目で親が暴走し、結果的に息子の自主性を奪ってダメ人間にしていく過程を笑うバカゲー。

## 2. コアループ（ゲームシステム）
プレイヤーは父親となり、次々と発生する息子のイベントに対して「Reigns風の2択スワイプ」で介入していく。

1. **イベント発生:** 画面中央にイベントカードが表示される。
2. **選択（スワイプ）:**
   * 左スワイプ: まともな親の対応（見守る、諭すなど）
   * 右スワイプ: りそうのちちおや（モンペ）の対応（金でもみ消す、権力行使、過剰防衛など）
   * ※スワイプ中、画面上部の4つのパラメーターアイコン上に影響度（大小の丸）をプレビュー表示する。
3. **結果表示:** 選択確定後、カードが飛び、結果のギャグテキスト（とパラメーター増減）がポップアップ表示される。
4. **フェーズ進行:** 一定枚数のカードをこなすと息子の年齢が上がり、最終的に社会人フェーズでエンディングを迎える。

## 3. パラメーター仕様
以下の4つのパラメーター（0〜100）を管理する。初期値はすべて50。
いずれかが 0 または 100 に達した場合、その時点で特殊なバッドエンドとなる。

| アイコン | パラメーター | キー | 上限/下限到達時の結末 |
|---|---|---|---|
| 🧠 | 自立心 | `independence` | 0 → 究極のパラサイトニートエンド / 100 → 息子が巣立ちエンド |
| 👑 | ワガママ度 | `ego` | 100 → 親への暴力・犯罪者エンド / 0 → 意志なし人形エンド |
| 💰 | 親の財力 | `wealth` | 0 → 自己破産エンド |
| 👁️ | 世間のヘイト | `hate` | 100 → 警察介入・社会的抹殺エンド |

independence=100 かつ ego=0 の同時達成は特別エンド（りそうのにんぎょう）へ優先分岐する。

## 4. フェーズ進行

全フェーズでイベントカードは合計25枚。各フェーズ5枚消化で次フェーズへ進む。
エンドはバッドエンドのみ（ノーマルエンド・グッドエンドは存在しない）。

| フェーズ | カード枚数 | 備考 |
|---|---|---|
| 幼児 | 5枚 | — |
| 小学校 | 5枚 | — |
| 中学校 | 5枚 | — |
| 高校 | 5枚 | — |
| 社会人 | 5枚 | 最終フェーズ。最終カードでエンディング分岐 |

## 5. エンディング一覧

エンドはすべてバッドエンド。途中でいずれかのパラメーターが限界値に達するか、社会人フェーズ終了時のパラメーター状態で分岐する。

| # | エンド名 | ending_id | 発生条件 | タイトル |
|---|---|---|---|---|
| 1 | 複合エンド | `end_doll` | `independence` = 100 かつ `ego` = 0 | りそうのにんぎょう |
| 2 | 🧠 巣立ちエンド | `end_independent` | `independence` = 100 | りそうのたびだち |
| 3 | 👑 人形エンド | `end_broken` | `ego` = 0 | りそうのひと |
| 4 | 🧠 ニートエンド | `end_neet` | `independence` = 0 | りそうのすみか |
| 5 | 👑 犯罪者エンド | `end_criminal` | `ego` = 100 | モンスターの産声 |
| 6 | 💰 自己破産エンド | `end_bankrupt` | `wealth` = 0 | りそうのだいしょう |
| 7 | 👁️ 社会的抹殺エンド | `end_cancelled` | `hate` = 100 | 炎上家族 |

エンディングデータは `data/endings.json` に格納。判定は `_check_bad_end()` で上記の順に elif チェックする（複合条件が最優先）。

### 各エンディングの到達パス（バランス調整 v3 + 7エンド実装時点）

| エンド | 推奨スワイプパターン | 発動カード |
|---|---|---|
| end_doll | **全左スワイプ** | elem_05（independence=100, ego=0 同時達成） |
| end_independent | 左多め（ego を 0 にしない混合） | DP 到達可 |
| end_broken | 左多め（independence を 100 にしない混合） | DP 到達可 |
| end_neet | **全右スワイプ** | jh_05（independence=0） |
| end_criminal | **全右 / jh_05 のみ左** | hs_03（ego=100） |
| end_cancelled | **inf/elem/hs/adult 右・jh 左** | hs_04（hate=100） |
| end_bankrupt | **全左（完走）** ※end_doll が先に発火するため get_final_ending 経由は困難 | DP 到達可 |

### 社会人フェーズクリア時の分岐ルール

25枚すべてをこなしてもパラメーターが限界値に達しなかった場合、**その時点で限界値に最も近いパラメーター**に応じてエンディングへ分岐する。

| 条件 | 分岐先 |
|---|---|
| `independence` が最低値 | 🧠 ニートエンド |
| `ego` が最高値 | 👑 犯罪者エンド |
| `wealth` が最低値 | 💰 自己破産エンド |
| `hate` が最高値 | 👁️ 社会的抹殺エンド |

同率の場合は上記の優先順位で決定。

## 6. シナリオデータ構造 (JSON形式)
各イベントカードは以下のJSONスキーマに従って定義し、Godot側でパースして使用する（またはCustom Resourceに変換する）。

```json
{
  "card_id": "unique_id",
  "phase": "小学校",
  "character_image": "image_path.png",
  "event_text": "イベントの状況説明文",
  "swipe_left": {
    "action_text": "まともな選択肢の短いテキスト",
    "result_text": "選択後の結果・ギャグ描写",
    "stat_changes": {
      "independence": 10,
      "ego": -5,
      "wealth": 0,
      "hate": 0
    }
  },
  "swipe_right": {
    "action_text": "モンペ選択肢の短いテキスト",
    "result_text": "過干渉によるギャグ描写と息子への悪影響",
    "stat_changes": {
      "independence": -15,
      "ego": 20,
      "wealth": -10,
      "hate": 30
    }
  }
}
```

`stat_changes` の値は -100〜+100。
シナリオデータは `res://data/cards/` 以下に配置する。

### テキストトーン方針
- **じわじわエスカレート系** — 幼児フェーズは比較的穏やか、社会人フェーズに近づくほど被害が大きくなる
- **末路強調スタイル** — 右スワイプの `result_text` は必ず息子・父への具体的な悪影響で締める
- **語り口** — フェーズ・カードに応じて客観ナレーション／息子の声／父の内心を使い分ける

### stat_changes フェーズ別設計指針
右スワイプ1回あたりの変動幅の目安。序盤は小さく、後半ほど大きくする。
※バランス調整済み（v3）。DP全探索（2²⁵通り）で4エンド全到達パスを確認済み。

| フェーズ | hate 変動幅（目安） | independence 変動幅（目安） | ego 変動幅（目安） |
|---|---|---|---|
| 幼児 | +2 | 0〜-2 | 0〜+5 |
| 小学校 | +3〜+8 | -2〜-5 | +3〜+5 |
| 中学校 | **+1〜+5** | -4〜**-7** | +3〜+5 |
| 高校 | 0〜**+3** | -4〜-7 | +3〜+8 |
| 社会人 | +7〜+10 | -5〜-9 | +3〜+8 |

左スワイプの `independence` 補填は全カード最大 **+5** に統一（混合プレイ時に independence が 100 に張り付かないよう調整）。

## 7. ビジュアル方針

Gemini API + rembgで生成するキャラクター素材の共通スタイル。

| 項目 | 設定 |
|---|---|
| スタイル | アニメ調2Dイラスト（manga-inspired） |
| アイコンサイズ | 32×32px（Godot上で2倍拡大し64×64で表示） |
| キャラクター生成ツール | Gemini API（`gemini-3.1-flash-image-preview`）+ rembg背景除去。生成スクリプト: `scripts/generate_char_images.py` |
| キャラクターサイズ | 縦400px基準・アスペクト比維持リサイズ・透過PNG（rembg処理済み） |
| キャラクター保存先（息子） | `assets/images/cards/[phase]/[card_id].png` |
| キャラクター保存先（父） | `assets/images/characters/father.png` |
| カード背景サイズ | 640×400px（Gemini API生成後、中央クロップ＋リサイズ済み。Godot上で`STRETCH_KEEP_ASPECT_COVERED`で表示） |
| カード背景生成ツール | Gemini API（`gemini-3.1-flash-image-preview`）。生成スクリプト: `scripts/generate_card_bgs.py` |
| カード背景保存先 | `assets/images/ui/card_bgs/[card_id].png`（card_idファイル未存在時はフェーズ背景にフォールバック） |
| 保存先 | `assets/images/` 以下 |

## 9. 技術仕様

### 基本設定

| 項目 | 内容 |
|---|---|
| エンジン | Godot Engine 4.x |
| エクスポート | Web（HTML5）/ WebGL 2.0（Compatibilityレンダラー） |

### 投稿先プラットフォームと制約

| プラットフォーム | 制約・注意点 |
|---|---|
| **Plicy** | GodotはiframeでのHP埋め込み不可。「大画面表示のみ」設定を使用。エクスポートHTMLを `index.html` にリネームしてzip化して投稿。 |
| **GodotPlayer** | SharedArrayBuffer対応済み。index.htmlリネーム不要。登録時にブラウザサイズを入力。 |

> ⚠️ Godot 4 WebエクスポートはCompatibilityレンダラー（WebGL 2.0）のみ。Safariでは動作不安定なためChromeを推奨。

### 画面解像度設定

| 項目 | 設定値 |

| アスペクト比 | `expand`（PCの横長モニターで上下に余白追加、破綻しない） |

### 主要スクリプト方針

| スクリプト | 役割 | 主なAPI |

#### カード反転アニメーション（Tween）
```gdscript
# scale:x を 1→0→1 で切り替え（2Dカードフリップ）
var tween = create_tween()
tween.tween_property(card, "scale:x", 0.0, 0.15).set_trans(Tween.TRANS_SINE)
await tween.finished
card_label.text = result_text  # scale=0の瞬間にテキスト差し替え
var tween2 = create_tween()
tween2.tween_property(card, "scale:x", 1.0, 0.15).set_trans(Tween.TRANS_SINE)
```

## 11. BGM・SE仕様

出力先: `assets/audio/`

- **BGM**: DOVA-SYNDROME（https://dova-s.jp/）からフリー素材を手動入手（ElevenLabs `compose_music` は有料プラン必須のため）
- **SE**: ElevenLabs MCP（`text_to_sound_effects`）で生成済み

### BGM方針

**テイスト:** 序盤から暗めのトーンで統一し、フェーズが進むにつれ不穏さ・絶望感が増していく。バカゲーの笑いの裏に漂う風刺と闇を音で演出する。

| フェーズ | ファイル名 | 曲名 | 作曲者 | テイスト |
|---|---|---|---|---|
| 幼児 | `bgm_infant.mp3` | 時計図書館 | MATSU | オルゴール調・不穏な可愛さ |
| 小学校 | `bgm_elementary.mp3` | Honny Trap | Phalene | ダーク・スウィングジャズ |
| 中学校 | `bgm_junior_high.mp3` | 都市伝説 | shimtone | ベル・不気味・都市伝説風 |
| 高校 | `bgm_high_school.mp3` | 追憶のプライベートアイ | 田中芳典 | シネマ風・重厚な暗さ |
| 社会人 | `bgm_adult.mp3` | 残滓念 | かずち | アンビエント・ホラー・絶望 |

**クレジット:** DOVA-SYNDROME標準ライセンスにより表記不要。都市伝説（shimtone）のみ任意でクレジット推奨。

### SE一覧

| 役割 | ファイル名 | タイミング | 音のイメージ |
|---|---|---|---|
| カードスワイプ | `se_swipe.mp3` | カードを左右にドラッグしたとき | 紙をめくる・滑らせる音 |
| パラメーター上昇 | `se_param_up.mp3` | パラメーターが増加したとき | 軽やかな上昇音（チーン） |
| パラメーター下降 | `se_param_down.mp3` | パラメーターが減少したとき | 重めの下降音（ズン） |
| 結果テキスト表示 | `se_popup.mp3` | 結果ポップアップが出るとき | ポンと出る軽い効果音 |
| バッドエンド突入 | `se_bad_end.mp3` | いずれかのパラメーターが限界値に達したとき | ドーンという重い演出音 |
